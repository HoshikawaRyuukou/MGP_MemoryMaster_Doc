- System : 對於開發者來說，應用程式使用了什麼架構/如何分割模組

## Product
- 為何設計此 app
- 功能總覽

## System

- Feature 功能是系統的那些直接滿足某些用戶需求的行為 
  - Lobby
  - ClassicGame
  - BattleGame
  - CollectionAlbum
  - SupportMe
  - Credit
